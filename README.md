Edge cases

Zero count - then not shown in result.

Input can be lower and uppercase (in the string)

```

Output: "Green: 2\nAmber: 1\nRed: 2"

Input: ""

Input: "Green,Dave,Whimsy,Red"

Input: "Green"
Output: "Green: 1"

Input: "Green, Green"
Output: "Green: 2"

Input: "Green, Red"

```
